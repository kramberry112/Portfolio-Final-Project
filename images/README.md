# Image Placeholders

This folder should contain the following images for your portfolio:

## Required Images:

### 1. profile.jpg
- **Size**: 350x350 pixels (square aspect ratio)
- **Description**: Your professional headshot or portrait
- **Usage**: About Me section
- **Tips**: 
  - Use a high-quality photo with good lighting
  - Professional appearance (business casual or formal)
  - Clear background preferred
  - Smile and make eye contact with the camera

### 2. project1.jpg
- **Size**: 600x400 pixels (3:2 aspect ratio)
- **Description**: Screenshot of your first project
- **Usage**: Projects section - Responsive Landing Page
- **Tips**:
  - Take a clean screenshot of your project
  - Show the most attractive part of the interface
  - Ensure text is readable in the screenshot

### 3. project2.jpg
- **Size**: 600x400 pixels (3:2 aspect ratio)  
- **Description**: Screenshot of your second project
- **Usage**: Projects section - Task Management App
- **Tips**:
  - Capture the main interface or dashboard
  - Show the app in use with some sample data
  - Highlight key features visually

### 4. project3.jpg
- **Size**: 600x400 pixels (3:2 aspect ratio)
- **Description**: Screenshot of your third project  
- **Usage**: Projects section - Weather Dashboard
- **Tips**:
  - Show the weather interface with sample data
  - Capture a visually appealing layout
  - Include charts or graphics if available

## Image Optimization Tips:

1. **File Size**: Keep images under 500KB for web performance
2. **Format**: Use JPG for photos, PNG for graphics with transparency
3. **Quality**: Optimize for web (85-90% quality for JPG)
4. **Alt Text**: The HTML already includes proper alt attributes
5. **Responsive**: Images will automatically scale on different devices

## Tools for Creating/Editing Images:

- **Free Online**: Canva, GIMP, Photopea
- **Screenshot Tools**: Built-in OS tools, Snipping Tool (Windows), Screenshot (Mac)
- **Image Optimization**: TinyPNG, ImageOptim, Squoosh
